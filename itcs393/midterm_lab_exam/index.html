    • 5. Show all teaching assignment information including course code, section, instructor ID, instructor’s first name and last name, the number of hours in lecture and lab, and its teaching hour in total. Sort the result by course code and section respectively
      Note: teaching hour in total =
      hour_lect + hour_lab
      (Expected: 6 columns, 259 rows)
      SELECT course_code, sec, a.instructorID, i.firstName, i.lastName, hour_lect, hour_lab, (hour_lect+hour_lab) AS teaching_hour FROM assignments a
INNER JOIN instructors i ON a.instructorID = i.instructorID ORDER BY course_code ASC, sec ASC;

    • 6. List all information about the courses beginning with “Special Topic” (Expected: all 6 columns of the courses table, 5 rows)

    • 8. Count the number of instructors per instructor’s title, sorted by the highest to lowest number (Expected: 2 columns, 5 rows)
      SELECT title, COUNT(title) AS numOfInstructors FROM instructors

    • 9. List course_code, course_name, program_code, and degree for the courses names containing the word “Database”. Sort the result by program_code ascendingly (Expected: 4 columns, 7 rows)
      SELECT course_code, course_name, c.program_code, p.degree FROM courses c
INNER JOIN programs p ON c.program_code = p.program_code
WHERE course_name LIKE "Database%" ORDER BY p.program_code ASC;
    • 10. List course_code, course_name, area_description for all “Free Elective” course types. Sort the result by course_code ascendingly (Expected: 3 columns, 4 rows)
      SELECT course_code, course_name, a.area_description FROM courses c
INNER JOIN areas a ON c.course_area = a.course_area
INNER JOIN coursetypes t ON c.type_code = t.type_code
WHERE course_type = "Free Elective" ORDER BY course_code ASC;
    • 11. List all information about Areas that have the number of courses in that area more than 12 courses. Sort the result from the highest number of course to the lowest one. (Expected: 3 columns, 4 rows)
      SELECT c.course_area, a.area_description, COUNT(program_code) AS numOfCourses FROM courses c
INNER JOIN areas a ON c.course_area = a.course_area
GROUP BY c.course_area HAVING numOfCourses > 12 
ORDER BY numOfCourses DESC;
    • 12. List the instructors who do not have the teaching assignment, i.e., no course to teach (Expected: 3 columns, 2 rows)
      SELECT firstName, lastName, COUNT(course_code) AS numcourse FROM instructors i
LEFT OUTER JOIN assignments a ON i.instructorID = a.instructorID
GROUP BY firstName,lastname HAVING numcourse = 0;

    • 13. Find the total number of courses per course type of all programs at the graduate level in the Semester 2 (Expected: 3 columns, 2 rows)
      SELECT c.type_code, t.course_type, COUNT(course_code) AS numOfCourses FROM courses c
INNER JOIN coursetypes t ON c.type_code = t.type_code
INNER JOIN programs p ON c.program_code = p.program_code
WHERE prog_level = "graduate" AND c.semester = 2
GROUP BY c.type_code;

    • 15. List the instructors who teach more than average total number of teaching hours. Show the instructor’s title, first name, last name, and the number of teaching hours from the highest to lowest. (Expected: 4 columns, 21 rows)
      SELECT title, firstName, lastName, SUM(hour_lect+hour_lab) AS totalTeachingHours FROM instructors i
INNER JOIN assignments a ON i.instructorID = a.instructorID
GROUP BY title,firstName,lastname HAVING totalTeachingHours >
(SELECT AVG(t.total) FROM
	(SELECT SUM(hour_lect+hour_lab) AS total FROM assignments
		GROUP BY instructorID
	) t
) ORDER BY totalTeachingHours DESC;
