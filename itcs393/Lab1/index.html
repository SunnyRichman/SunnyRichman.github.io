  1. Write the SQL statements to create two user accounts hosted in the local host with the following details:
    • Account 1
      • Username: student_xxx where xxx is the last 3 digits of your student ID
      • Password: abc123*
    • Account 2
      • Username: student_yyy where yyy is the last 3 digits of your friend ID
      • Password: abc123*
    CREATE USER IF NOT EXISTS 'student_132'@'localhost'
IDENTIFIED BY 'abc123*';
CREATE USER IF NOT EXISTS 'student_133'@'localhost'
IDENTIFIED BY 'abc123*';
  1. Write the SQL statements to create a user role, named itcs393_students hosted in the local host and assign the global database-level privilege, include CREATE, ALTER, DROP, INSERT, UPDATE, and DELETE.

    💡: Use . to apply all databases and all tables.

CREATE ROLE 'itcs393_students'@'localhost';
GRANT CREATE, ALTER, DROP, INSERT, UPDATE, DELETE ON *.* TO 'itcs393_students'@'localhost';
  1. Write the SQL statements to assign a role, named itcs393_students created in 2. To both user accounts, student_xxx and student_yyy, created in 1.
GRANT 'itcs393_students'@'localhost' TO 'student_132'@'localhost', 'student_133'@'localhost';
