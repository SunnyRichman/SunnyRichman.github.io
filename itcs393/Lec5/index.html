Aggregate Functions

  • Aggregate functions are used to summarize data

COUNT

Count number of not-null values of an attribute

MAX

Find highest value in a table

MIN

Find lowest value in a table

SUM

Compute total sum for any specified attribute

AVG

Compute the average value of any specified attribute

STDDEV

Compute the population standard deviation

Aggregate Function: COUNT

  • Used to find the total number not-null values of an attribute.
  • List all managers in the company
SELECT * FROM Employees WHERE jobTitle LIKE '%Manager%';
  • How many employee who are managers?
SELECT COUNT(*) AS numManager FROM Employees WHERE jobTitle LIKE '%Manager%';
  • Can be used in conjunction with the DISTINCT (selecting non-duplicate value) clause
SELECT DISTINCT productVendor FROM Products;
SELECT COUNT(DISTINCT productVendor) AS numVendor FROM Products;

Aggregate Function: MAX, MIN, SUM, AVG

Find the total, minimum, average, and maximum amount of payments of this company in the year 2004.

SELECT SUM(amount), MIN(amount), AVG(amount), MAX(amount) FROM Payments WHERE YEAR(paymentDate)=2004;

Aggregate Function: STDDEV

  • Standard deviation is a statistical measure of diversity or variability in a data set.
    • Low SD means data points are closed to the mean
    • High SD indicates greater variability in data points

What is the average and standard deviation of MSRP of the "Classic Cars"?

SELECT AVG(MSRP), FORMAT(STDDEV(MSRP),2) FROM Products WHERE productLine='Classic Cars';

Aggregate Function: GROUP BY

  • The aggregate functions can be used to summarize data based on group
    • Example: How many products per vendor?
    SELECT * FROM products ORDER BY productVendor ASC;
  • GROUP BY: Group rows into smaller collections by given columns, then, apply aggregate functions to summarize the data in each group.
  • 💡: the GROUP BY clause is valid only when used in conjunction with one of the SQL aggregate functions
SELECT productVendor, COUNT(productCode) as numProd FROM Products GROUP BY productVendor ORDER BY productVendor;

GROUP BY with HAVING

  • The HAVING clause operates like the WHERE clause of the GROUP BY clause
  • Only applied to the output of a GROUP BY operation
  • Example: What vendors providing exactly 8 products?
SELECT productVendor, COUNT(productCode) as numProd FROM Products 
GROUP BY productVendor HAVING numProd = 8 ORDER BY productVendor;
